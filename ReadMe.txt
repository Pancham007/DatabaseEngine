---------------------- DAVISBASE--------------------
• Built a rudimentary database engine that is loosely based on a hybrid between MySQL and SQLite using Java
• Implementation should operate entirely from the command line and API calls (no GUI)
----------------------------------------------------------------------------------------------------------------------------------

README

1. Open the project in Eclipse or in command prompt (if Java_Home and ClassPath variables are initialized.)
2. Run DavisBase.java as a Java Application.
3. Choose if you want to use the default prompt (option N) or a custom prompt (option Y).
4. If you choose to use a custom prompt, just write the prompt name like "command_me".
5. Enter the commands after the prompt (e.g. "davisql>") appears.
6. Exit the application by executing the "EXIT;" command.
